# Todo Stack

Full-stack todo application with a Python API backend and a Next.js frontend.

 Project Structure
* `/backend` - Python server and SQLite database
* `/frontend` - Next.js client application

 Setup

 Backend
1. Navigate to the folder: `cd backend`
2. Set up virtual environment: `python3 -m venv venv && source venv/bin/activate`
3. Run the server: `fastapi dev`

 Frontend
1. Navigate to the folder: `cd frontend`
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
